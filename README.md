# coral

Coral is a deep AI agent, with a Discord integration.

## Installation

```
$ gh repo clone uukelele/coral
$ cd coral
$ pip install -e . -U
```

### Updating

```
$ cd coral
$ git pull
```

## Usage

```
$ coral create my-bot
$ cd my-bot
$ # Edit the `config.yaml` and `config.md.j2` files in a text editor.
$ coral run
```
